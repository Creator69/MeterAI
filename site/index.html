01

Why agents can't pay for APIs today

Subscription + credit card signup Void Assumes a human fills out a form. Agents can't.
Fixed monthly pricing tier Void An agent needs 14 calls today, not a $49/mo plan.
Pre-established trust / manual API key Void Providers can't safely serve anonymous agents without it.
HTTP 402 + USDC nanopayment on Arc Settled Exact usage, exact price, paid and verified per request.
02

How it works

1

Provider wraps an endpoint with MeterAI and sets a USDC price per call.

2

Agent requests it unpaid, gets back HTTP 402 with the price.

3

Agent's wallet authorizes a USDC nanopayment on Arc, gas-sponsored by Paymaster.

4

Agent retries with payment proof, gets served — no human anywhere in the loop.
